Introduction

C# build with .NET(Core) is a cross-platform, open-source,re-implementation of .NET framework which is actively maintained by Microsoft and a community of Github Developers which can be used on both Windows and Linux systems. C# coding is done in an object-oriented fashion. C# is a High Level Language build with a compiler(runs the entire program as input instead of line by line).

There are a few benefit to using C# with .NET Core:

  1. Linux developers can try C# and .NET business without a Windows License
  2. If you would like to create a micro-service but don't want to use Node.js or other javascript -on-the-server frameworks.
  3. Take a pre-existing MVC app or written in .NET framework and run it on a Linux platform.
  4. You can have the "Write once, build anywhere" that Java provides but with the C# platform.

.NET Core Supports the below:

  1. Console Applications
  2. Web applications (asp.NET, Webapps, MVC)
  3. Class Libraries
  4. Unit Tests

C# and .NET Framework: An Object Oriented Programming Language

An Object Oriented language stipulates that the variable declarations and inputs are declared in the main method.

Any processing of the actually varaibles are done in 'Methods', A method is a code block that contains a series of statements. A program causes the statements to be executed by calling the method and specifying any required method arguments. In C#, every executed instruction is performed in the context of a method.

Examples of this include declaring number variables in the main subset of the program, and having methods declared to calculate the sum, multiplications and division of manipulating those numbers.

Variables

Variables: A variable is a name given to a storage that our program can manipulate.

Declaring variables

When declaring a variable you will always:

  • First word is the the data type of your variable (example number, string, real, boolean)
  • Your second word would be the name of your variable, variable naming has a few conventions, but my preffered declaration is Camel Case, where your first word is lower case and the other words start with an Upper Case letter. There are no spaces in the variable name. (example: firstNum, secondNum)
  • You will then initiate your variable, "=" with the relevent value for that variable type.

Example: int x=1;

String name="John";

Variable scope

The different types of variables are as follows:

  • int: used to store integer numbers (eg. 3 or 2000)
  • float: used to store floating point numbers (eg. 3.5 or 9.8)
  • double: used to store floating point numbers (eg. 3.567 or 5.909)
  • decimal: Used to store monetary values (R200,00 or R59,99)
  • String: Used to store Text "John", "Tom"
  • bool: a boolean value is either true or false (0,1)
  • char: Stores one character or letter. Eg: "A", "C"
Global variables

Global variables are variables that can be used within the entire program. C# does not support global variables, however this can be catered too via using a static class where you declare the static variables.

Constants

A constant variable is a variable thats value will never change. A good example of this is Pi or Gravity, these values will never change.
