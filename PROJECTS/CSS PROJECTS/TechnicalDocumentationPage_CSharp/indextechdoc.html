Introduction

C# build with .NET(Core) is a cross-platform, open-source,re-implementation of .NET framework which is actively maintained by Microsoft and a community of Github Developers which can be used on both Windows and Linux systems. C# coding is done in an object-oriented fashion. C# is a High Level Language build with a compiler(runs the entire program as input instead of line by line).

There are a few benefit to using C# with .NET Core:

  1. Linux developers vsn try C# and .NET business without a Windows License
  2. If you would like to create a micro-service but don't want to use Node.js or other javascript -on-the-server frameworks.
  3. Take a pre-existing MVC app or written in .NET framework and run it on a Linux platform.
  4. You can have the "Write once, build anywhere" that Java provides but with the C# platform.

.NET Core Supports the below:

  1. Console Applications
  2. Web applications (asp.NET, Webapps, MVC)
  3. Class Libraries
  4. Unit Tests
